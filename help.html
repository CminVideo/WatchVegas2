WatchVegas2 Help Guide

Introduction

WatchVegas2 is a powerful monitoring tool designed to help you oversee your system’s hardware performance and Vegas Pro plugins. It provides real-time graphs, alerts, and detailed logs to ensure your system runs smoothly.

Features Overview

  • Hardware Monitoring: Tracks CPU, GPU, VRAM, RAM, Disk usage, and CPU temperature with graphical charts.
  • Vegas Pro Module Scanning: Monitors loaded plugins and system modules within Vegas Pro.
  • Watched Plugins: Notifies you when specific plugins you care about are loaded.
  • Interference Application Monitoring: Detects potentially conflicting background applications.
  • Logging and Reporting: Saves detailed logs and crash snapshots for troubleshooting.
  • Automatic and Manual Screenshots: Capture screenshots on alerts or manually for documentation.

Watched Plugins Explained

The Watched Plugins feature lets you specify plugin names or keywords that are important to monitor. When a plugin matching one of these keywords is detected, WatchVegas2:

  • Logs a special [WATCHED] message in the application log and saves it to the disk log.
  • Optionally triggers a screenshot capture to document the state of your system.

Example:

[2024-06-09 14:25:30] [WATCHED]: BorisFXPlugin.dll (matched 'Boris')

This helps you keep track of critical or suspicious plugins easily.

How to Use Watched Plugins

  • Open the settings window.
  • Add or remove plugin keywords in the Watched Plugins list.
  • Enable logging and/or screenshot triggers for watched plugins.
  • Save the settings to apply changes.

Interference Application Monitoring

WatchVegas2 also monitors for known background applications that might interfere with Vegas Pro, like Discord, OBS, and Steam. If detected, it alerts you with warnings in the log.

Summary Report

The Summary Report feature provides a quick snapshot of your system and Vegas Pro status at any moment. Accessible from the Tools > Summary Report menu, it displays key information such as:

  • System Load Level: Current overall system load with color indication.
  • CPU, GPU, VRAM, and Disk Usage: The latest usage percentages of your hardware.
  • CPU Temperature: Current CPU temperature, including adaptive baseline if used.
  • Loaded Modules: Number of modules currently loaded into Vegas Pro and the most recently loaded module.
  • Watched Plugins: Lists plugin modules you have specified to watch that are currently active.
  • Conflict Monitor Status: Indicates whether interference monitoring is enabled, and lists any currently detected conflicting applications.
  • Heartbeat Logging: Shows if heartbeat monitoring is active to track ongoing system status.

This report is displayed directly within the main log window for convenient access and real-time review, without needing to open separate forms or windows. It is designed to help you quickly assess system health, plugin states, and potential conflicts.

Getting Started

  1. Launch WatchVegas2. Ensure only one instance is running.
  2. Configure your settings including watched plugins, alert thresholds, and interference monitoring.
  3. Start Vegas Pro and monitor the logs and hardware stats live.
  4. Use the Refresh and Filter options to control module scanning and log viewing.
